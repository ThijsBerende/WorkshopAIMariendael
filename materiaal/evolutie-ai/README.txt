Twee files die je kunt aanpassen zijn variables.py (in bestand zie je wat er aangepast kan worden)
en config.txt, in config kun je bv de populatie aanpassen per generatie.

run level_editor.py om een level te editen. Met je pijltjes op je toetsenbord kun je een level selecteren.
Als je op save drukt en de editor afsluit, dan slaat hij je level op als "levelx_data.csv".
Als je dit level wilt spelen, ga naar variables.py en pas daar de naam aan.

run game.py om het level zelf te spelen, met w om te springen, a voor links en d voor rechts.

run ai.py om de ai het level te laten spelen.
